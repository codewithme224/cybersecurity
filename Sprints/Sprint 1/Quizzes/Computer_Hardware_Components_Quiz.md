# Quiz: Computer Hardware Components

**Instructions:** Answer the following questions to test your understanding of the physical layer of cybersecurity.

### Question 1
An attacker uses a specialized tool to extract encryption keys from the RAM of a laptop that was recently powered down. What is this attack called?
- A) Buffer Overflow
- B) Cold Boot Attack
- C) MAC Spoofing
- D) UEFI Bootkit

### Question 2
Which component is responsible for the initial boot process and is a prime target for "Bootkits" that persist even after an OS reinstall?
- A) RAM
- B) SSD
- C) UEFI/BIOS
- D) CPU

### Question 3
What happens during a "Buffer Overflow" attack?
- A) The CPU overheats and shuts down.
- B) The NIC fails to connect to the network.
- C) Data exceeds the memory allocated in RAM, potentially allowing code execution.
- D) The hard drive runs out of storage space.

### Question 4
Why is Full Disk Encryption (FDE) important for storage devices?
- A) It makes the computer boot faster.
- B) It prevents the CPU from overheating.
- C) It ensures that data is unreadable if the physical drive is stolen.
- D) It prevents MAC spoofing.

---
**Answer Key:**
1: B | 2: C | 3: C | 4: C
