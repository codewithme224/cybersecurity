# Cybersecurity Learning Roadmap

## Sprint 1: The Foundation
- [ ] Computer Hardware Components
- [ ] Connection Types and their function
- [ ] OS-Independent Troubleshooting
- [ ] Basics of Popular Suites (Google, MS Office, etc.)
- [ ] Basics of Computer Networking (NFC, WiFi, BT, etc.)

## Sprint 2: The Attacker's Toolkit
- [ ] Operating Systems (Windows, Linux, MacOS)
- [ ] GUI vs CLI Navigation
- [ ] Permissions & CRUD
- [ ] Common CLI Commands
- [ ] Virtualization (VMWare, VirtualBox, Proxmox)
- [ ] Troubleshooting Tools (nslookup, iptables, ping, dig, nmap, etc.)

## Sprint 3: Security Core
- [ ] OSI Model & Common Protocols/Ports
- [ ] SSL/TLS Basics
- [ ] Subnetting, IP Terminology (CIDR, Gateway, etc.)
- [ ] VLAN, DMZ, ARP, DHCP, DNS, NAT
- [ ] CIA Triad
- [ ] Cryptography (Hashing, Salting, PKI, Keys)
- [ ] Defense in Depth & Zero Trust

## Sprint 4: Offensive Ops
- [ ] Common Hacking Tools & Exploit Frameworks
- [ ] Penetration Testing Rules of Engagement
- [ ] Web Based Attacks & OWASP Top 10
- [ ] Malware Types & Functionality
- [ ] Privilege Escalation
- [ ] Social Engineering (Phishing, etc.)
- [ ] Brute Force vs Password Spray

## Sprint 5: Defensive Ops & Cloud
- [ ] Forensics & Threat Hunting
- [ ] Vulnerability Management & Reverse Engineering
- [ ] Incident Response Process (Preparation $\rightarrow$ Lessons Learned)
- [ ] SIEM, SOAR, IDS/IPS, Honeypots
- [ ] OS Hardening & Endpoint Security
- [ ] Cloud Security Concepts (SaaS, PaaS, IaaS, Shared Responsibility)
- [ ] AWS, GCP, Azure Fundamentals

## Sprint 6: Specialized & Programming
- [ ] Programming (Python, Go, JavaScript, C++, Bash, PowerShell)
- [ ] Frameworks (ATT&CK, Kill Chain, Diamond Model)
- [ ] Standards (ISO, NIST, RMF, CIS)
- [ ] Certifications Path (CompTIA, OSCP, CISSP, etc.)
- [ ] CTF Practice (HackTheBox, TryHackMe)
